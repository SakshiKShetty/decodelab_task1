## Project Overview

Project 1 of the DecodeLabs Full Stack Development internship — the interface phase. The objective is to build a fully responsive frontend interface using HTML, CSS, and JavaScript, demonstrating core web fundamentals before backend integration.

---

## Requirements Met

| Requirement | Status |
|---|---|
| Pure HTML, CSS, JavaScript (no frameworks) | Done |
| Responsive layout for all screen sizes | Done |
| Semantic HTML5 landmark elements | Done |
| Mobile-first approach | Done |
| Clean and user-friendly UI | Done |
| WCAG accessibility compliance | Done |

---

## Tech Stack

- **HTML5** — Semantic landmarks (`header`, `nav`, `main`, `article`, `footer`)
- **CSS3** — Grid, Flexbox, custom properties, `clamp()` for fluid typography
- **JavaScript** — Vanilla JS, no external libraries

---

## Features

- Sticky responsive navbar with animated hamburger menu for mobile
- Hero section with browser mockup visual
- Features grid that adapts from 1 to 2 to 4 columns across breakpoints
- Stats section with dark background contrast
- Projects portfolio with 4-card responsive grid
- Contact form with client-side validation
- Back-to-top button with scroll detection
- Active nav link highlighting on scroll via IntersectionObserver
- Reduced motion support via `prefers-reduced-motion`
